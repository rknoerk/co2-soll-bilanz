## Automated Checkup — Vorgaben-Status

Das Checkup-Sheet führt automatisch Buch über den Status jeder Vorgabe:

| Spalte | Bedeutung |
|---|---|
| C (Eingehalten) | TRUE wenn als eingehalten markiert |
| D (Eingehalten, da nicht relevant) | TRUE wenn nicht anwendbar |
| E (Nicht eingehalten) | TRUE wenn nicht eingehalten |

Sonderfälle:
- I.1, I.4, I.5, IV.4, IV.5, V.2, V.10 → haben kein "nicht relevant"-Feld (Spalte D = "---")
- Auslandsanteil → eigene Zeile, nur TRUE/FALSE in C
